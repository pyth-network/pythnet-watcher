You can run the project using `cargo run` by passing the required flags:

```bash

  --pythnet-url wss://api2.pythnet.pyth.network \
  --secret-key /path/to/secret.key \
  --wormhole-pid H3fxXJ86ADW2PNuDDmZJg6mzTtPxkYCpNuQUTgmJ7AjU \
```
